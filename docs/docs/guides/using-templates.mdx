Using [templates](/docs/ref/templates) is the easiest way to get effects playing using the library. Calling them is simple, but some examples are shown here nevertheless.

## Confetti

A demonstration of the [confetti template](/docs/ref/templates#confetti).

<DemoCodeblock method={(party, e) => party.confetti(e, {
    count: party.variation.range(0, 100),
    size: party.variation.range(0.6, 1.4)
})}>

```js
const element = document.getElementById("demoConfetti");
party.confetti(element, {
    // Specify further (optional) configuration here.
    count: party.variation.range(0, 100),
    size: party.variation.range(0.6, 1.4),
});
```

</DemoCodeblock>
